FACTORS AFFECTING SIZE OF DIGITAL FILES

Introduction
When you are working with digital images, the size of the foles is important for a vriety of reasons.
Files occupy space on storage devices. Although storage devices are becoming cheaper whole holding more information, it is still true that larger files occupy more space. More space costs more money.
Related to the occupation of space is the fact that large files can be difficult totransmit electronically. The transmission may take an inordinately long time or even abort if the internet services are limited.
The quality of a digital image is an imporant factor that is influenced by the size of the file. In general, the larger the file the greater the quality of the image. Most cameras will produce ipeg files unless you specifically arrange for RAW files to be produced. Starting with the file from the camerad, you can reduce the size of the file for use in different situation. One could keep the original for printing and use files of reduced size for transmission to colleagues.
Whether an image is in colour or black and white influences the size of the file. Coloured images are formed by combining information from three channels, the red, green and blue channels. When the colour information is discarded, the black and white images that results is about one third the size of the coloured version.
The problem with reducing the size of a digital file is that with increased reduction in size comes degradation of the image. With excessive reduction of size, the file becomes essentially unprintable.

As you might expect, large digital files take more time to process than smaller ones. When you are used to fast action of computers, the slowing down can be frustrating. If a processing step normally takes one second, most people find it unacceptably long when the process takes ten seconds - even though ten seconds is really a very short time.
You need to be careful indealing wikth black land white images. If you scan a blackand white image with the scanner set for the colour mode, the scanner will treat theblack and white image as if it was a colour image and produce a large file.
  1. Colour mode
    Select an area of the picture to be scanned.
    1. Scan the area in the Color mode and save the image as a high quality file with the name Colour scan.
    2. Scan the same area in the Grayscale mode and save the image as a high quality file with the name Greyscale scan
    3. Scan the same area in the Black and White mode and save the image as a high quality file with the name Black and White.
  2. Size of image
    1. Scan the whole image and save the file as Original
    2. Scan only the sample lanes and save the file as Sample lanes.
    3. Scan only the standard lanes and save the file as Standards.
  3. Resolution
    1. Scan the sample lanes at a resolution of 300dpm and save the file as 300 dpm
    2. Scan the sample lanes at 72dpm and save the file as 72 dpm
  4. Determination of te size of the files
    1. Right click on the icon of each image
    2. Click on Properties
    3. You can now read off the size of each file
